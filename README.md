## Installation

This tool require [Node.js](https://nodejs.org/) to run.

Clone this package
```sh
git clone https://github.com/dtwhen/Inquirer-demo
```
Go to directory - install the dependencies - run this code
```sh
cd Inquirer-demo
npm i
node .
```
